# Michael / David Robles Portfolio

Portfolio bilingue para presentar experiencia practica en ciberseguridad, redes, Linux, Proxmox, DevOps, seguridad web y despliegue de aplicaciones.

## Stack

- Next.js
- React
- TypeScript
- AnimeJS
- Lucide React

## Desarrollo

```bash
npm install
npm run dev
```

La aplicacion corre localmente en `http://localhost:9023`.

## Produccion

```bash
npm run build
npm run start
```
