## Descripción

Este repositorio contiene el código fuente de mi portfolio técnico personal.  
El objetivo del proyecto es centralizar mi perfil profesional, mis proyectos, mis laboratorios, mis certificaciones, mis servicios publicados y mi experiencia práctica en áreas relacionadas con ciberseguridad, redes, Linux, servidores y desarrollo web.

Más que una página estática, este portfolio funciona como una carta de presentación técnica. Muestra proyectos reales desplegados, servicios administrados en homelab, prácticas de seguridad ofensiva y defensiva, laboratorios documentados y herramientas que utilizo en entornos reales.

---

## Enfoque Principal

Este portfolio está orientado a demostrar experiencia práctica en:

- Ciberseguridad ofensiva y defensiva.
- Administración de servidores Linux.
- Redes, routing, switching y servicios internos.
- Virtualización con Proxmox.
- Despliegue de aplicaciones web.
- Análisis de vulnerabilidades con herramientas SAST y DAST.
- Operación de servicios mediante Docker, Cloudflare Tunnel y dominios propios.
- Documentación técnica de laboratorios y proyectos reales.
- Desarrollo de aplicaciones modernas con React, Next.js y TypeScript.

## Proyectos Destacados

### RitmoHub

Plataforma social musical desarrollada con enfoque en comunidad, perfiles, sesiones, eventos, autenticación, recuperación de cuenta y base de datos PostgreSQL.

Características principales:

- Aplicación web moderna con Next.js, React y TypeScript.
- Sistema de autenticación y sesiones.
- Recuperación de cuenta.
- Despliegue real mediante dominio propio.
- Arquitectura pensada para crecimiento y operación en servidor.

---

### PPT del Terror

Juego web con sistema de login, registro, ranking semanal y global, persistencia de puntuaciones y panel administrativo para observar actividad técnica.

Características principales:

- Frontend desarrollado con React y Vite.
- Backend con Express.
- Ranking persistente.
- Controles de seguridad.
- Rate limiting.
- Despliegue en entorno real.

---

### CryptoToolbox

Herramienta orientada a criptografía básica, hashes, checksums, reputación, perfiles, chat y monitoreo administrativo.

Características principales:

- Generación y verificación de hashes.
- Soporte para MD5, SHA1 y SHA256.
- Verificación de archivos.
- Socket.IO.
- PostgreSQL.
- Panel administrativo para observar actividad.
